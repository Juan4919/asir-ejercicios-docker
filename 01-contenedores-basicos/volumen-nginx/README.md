# Despliegue de Nginx con volumen

Ejercicio para practicar volúmenes con Docker.

## Qué hace

Levanta un contenedor Nginx que sirve un HTML personalizado desde un volumen.

## Cómo usar
```bash
chmod +x comandos.sh
./comandos.sh
```

Luego abre http://localhost:8080

## Comandos útiles
```bash
docker ps              # ver contenedores
docker logs mi-nginx   # ver logs
docker stop mi-nginx   # parar el contenedor
docker rm mi-nginx     # eliminar el contenedor
```
