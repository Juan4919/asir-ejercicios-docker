# ASIR - Ejercicios Docker y contenedores

## Sobre este repositorio
Ejercicios prácticos de Docker realizados durante el ciclo ASIR (IES Muralla 2024-2026). 
Documentación de mi aprendizaje en Docker con contenedores, volúmenes, imágenes y redes.

## Tecnologías practicadas
- Docker Engine
- Contenedores Linux
- Volúmenes y persistencia de datos
- Gestión de imágenes
- Redes Docker

## Estructura del repositorio
```
asir-ejercicios-docker/
├── 01-contenedores-basicos/    # Primeros pasos con contenedores
├── 02-volumenes/               # Persistencia de datos con volúmenes
├── 03-redes/                   # Networking entre contenedores
└── 04-imagenes/                # Creación de imágenes personalizadas
```

## Entorno de trabajo
- **Sistema:** WSL2 Ubuntu en Windows
- **Docker:** Docker Engine
- **Control de versiones:** Git + GitHub

## Cómo usar este repositorio
Cada ejercicio contiene:
- **README.md**: Explicación del ejercicio
- **comandos.sh**: Script con los comandos ejecutados
- **Archivos necesarios**: Configs, HTML, etc.
